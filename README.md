Weather App 🌦️

A responsive weather forecast app built with React, fetching real-time weather data from OpenWeather API. 🚩

- 🌤️ Search weather by city name
- 📍 Detect location with Geolocation API
- 📱 Fully responsive UI
- 🔄 Real-time API integration
